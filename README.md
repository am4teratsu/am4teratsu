## Sobre mí

Estudiante de Ingeniería de Sistemas que disfruta construir interfaces que se vean bien y funcionen mejor.
Me importa el detalle visual, el código limpio y que la experiencia del usuario tenga sentido desde el primer clic.

Cuando no estoy codificando, estoy en alguna partida — y sí, algo de ese instinto competitivo se me pega cuando programo 🎮
